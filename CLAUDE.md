# vision-recovery-traning

視力回復トレーニングアプリ

## 概要

目のトレーニングを通じて視力回復をサポートするWebアプリケーション。HTML/CSS/JavaScriptのみで構築され、サーバー不要でブラウザ上で動作する。

## 技術スタック

- **HTML5** — 構造・マークアップ
- **CSS3** — スタイリング・アニメーション
- **JavaScript (Vanilla)** — ロジック・インタラクション
- ビルドツール・フレームワーク不使用

## ディレクトリ構成（想定）

```
vision-recovery-traning/
├── index.html          # エントリーポイント
├── css/
│   └── style.css       # スタイルシート
├── js/
│   └── main.js         # メインロジック
└── assets/             # 画像・音声などの静的リソース
```

## 開発方針

- 外部ライブラリは最小限に抑え、Vanilla JSを優先する
- モバイル・デスクトップ両対応のレスポンシブデザイン
- ユーザーの目に優しい配色・コントラストを考慮する
- アクセシビリティ（WCAG）に配慮した実装

## 実行方法

`index.html` をブラウザで直接開くか、ローカルサーバーで確認する。

```bash
# Python でローカルサーバーを起動する場合
python -m http.server 8080
```

## コーディング規約

- インデント: スペース2つ
- 文字コード: UTF-8
- JavaScriptはES6+を使用（`const`/`let`、アロー関数、モジュールなど）
- コメントは日本語で記述する
