# 視力回復トレーニング

目のトレーニングを通じて視力回復をサポートするWebアプリケーションです。

🔗 **[アプリを開く](https://fc-kawaguchi.github.io/vision-recovery-traning/)**

---

## 機能

### 👁 視力測定
- ランドルト環（Cマーク）を使った視力測定
- 右目・左目・両目の3パターンで測定
- 画面との距離・モニターDPI・測定範囲を設定可能
- 測定結果を視力数値で表示、距離別の推定視力早見表も表示
- キーボード（↑↓←→）でも回答可能

### 🎯 ピント調整トレーニング
- 近くと遠くを交互に見てピント調整力を鍛える
- 遠見フェーズは2つの○を1つに重ねる立体視トレーニング
- 難易度3段階（かんたん / ふつう / むずかしい）

### 🔄 眼筋トレーニング
- 点が現れた方向を素早く判断して眼球を動かすトレーニング
- キーボード（↑↓←→）でも回答可能
- 難易度3段階（かんたん / ふつう / むずかしい）

### ⚫ トラッキングトレーニング
- 動くボールを目で追い、赤く光ったら素早くタップ
- ボールはランダムにカーブ・Uターンする
- クリック / タップ / Enterキーで反応
- 難易度3段階（かんたん / ふつう / むずかしい）

---

## 使い方

1. [アプリを開く](https://fc-kawaguchi.github.io/vision-recovery-traning/)
2. メニューから「視力測定」または「トレーニング」を選択
3. 設定を選んで「スタート」（Enterキーでも開始可能）
4. トレーニング中はスペースキーで一時停止

## ローカルでの実行方法

```bash
# リポジトリをクローン
git clone https://github.com/fc-kawaguchi/vision-recovery-traning.git
cd vision-recovery-traning

# index.html をブラウザで開くだけでOK
# または Python でローカルサーバーを起動
python -m http.server 8080
```

---

## 注意事項

- 視力測定の結果はあくまで参考値です。正確な視力測定は眼科を受診してください。
- 目に痛みや異常を感じた場合はすぐに使用を中止してください。
